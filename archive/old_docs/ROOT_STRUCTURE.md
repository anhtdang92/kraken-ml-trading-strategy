# 📁 Root Directory Structure
## Crypto ML Trading Dashboard - Clean Organization

### 🎯 **Core Application Files**
- `app.py` - Main Streamlit application
- `requirements.txt` - Python dependencies
- `README.md` - Project overview and quick start
- `PROJECT_INDEX.md` - Complete file organization guide

### 📁 **Organized Directories**
- `docs/` - Complete documentation suite
- `scripts/` - Automation and utility scripts
- `ml/` - Machine learning components
- `data/` - Data handling and API integration
- `gcp/` - Google Cloud Platform integration
- `config/` - Configuration and secrets
- `tests/` - Test suites (unit and integration)
- `examples/` - Reference code and examples
- `assets/` - Static assets (images, icons, styles)
- `tools/` - Development and deployment tools
- `infrastructure/` - Infrastructure as code
- `monitoring/` - Logs, metrics, and dashboards
- `backup/` - Backup scripts and data
- `deployment/` - Deployment configurations

### 🔧 **Development Files**
- `.gitignore` - Git ignore rules
- `.env` - Environment variables template
- `LICENSE` - Project license
- `CHANGELOG.md` - Project changelog

### 🚀 **Cloud Functions**
- `cloud_functions/` - Serverless functions

### 📦 **Runtime Directories**
- `models/` - Trained ML models
- `venv/` - Python virtual environment
- `__pycache__/` - Python cache files
